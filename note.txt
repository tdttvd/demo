Branching & Merging

Branching:
  Show all branches:
    git branch
  Create new branch:
    git checkout -b <branch>
  Change working branch:
    git checkout <branch>
  Delete a branch:
    git branch -d <branch>

Merging: Merge A <-- B (Merge B into A)
  Step 1: Change working branch to A
    git branch A
  Step 2: Merge A <-- B
    git merge B
